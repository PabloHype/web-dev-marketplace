# Web Development Marketplace

A modern marketplace platform to showcase and sell web development services.

## Features
- Service listings with pricing
- Portfolio showcase
- Contact form for inquiries
- Responsive design
- Mobile-friendly interface

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize with your services and portfolio

## Hosting

This site is ready for GitHub Pages. Enable it in repository settings.
